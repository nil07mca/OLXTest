# OLXTest
 To use the code install AFNetworking using cocoapod .
 
 Please add the following in pod file to install AFNetworking
 
************************************************
source 'https://github.com/CocoaPods/Specs.git'
platform :ios, '8.0'

target 'OLXTest' do
pod 'AFNetworking', '~> 3.0'
end
*************************************************
